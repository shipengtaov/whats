whats
========

Search keyword use Google, Print first link's text content

Usage
-------

	$ whats '美丽新世界'


Install
-------

	$ pip install whats

Proxy
-----

shadowsocks proxy example:

	$ export http_proxy='socks5h://127.0.0.1:1080'
	$ export https_proxy='socks5h://127.0.0.1:1080'
	$ whats '美丽新世界'
